# Simple Portfolio Design

## Clone The Repository

```sh
git clone https://github.com/nousantx/Portfolio-pug-design
```

## Install Dependencies

```sh
npm i
```

or :

```sh
yarn
```

## Start Project

```sh
npm run dev
```

or

```sh
yarn dev
```
