# Simple Portfolio Design

A simple template for portfolio design using oug and tenoxui.

## Clone The Repository

```sh
git clone https://github.com/nousantx/Portfolio-pug-design
```

## Install Dependencies

```sh
npm i
```

or :

```sh
yarn
```

## Start Project

```sh
npm run dev
```

or

```sh
yarn dev
```
